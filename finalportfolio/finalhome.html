Recurring Dream

The recurring dream project was the first homework assignment we had in Web Design class. We learned about h1 and p elements.

All About Me

This was our next homework assignment. This website displays some facts and basic information about me. We learned how to insert images, lists, and videos into our websites. This was also the first time I made style changes such as color and font.

My Favorite Animal - Beluga

This project displays some information and fun facts on my favorite animal: Belugas. This was the first project we did practicing using a CSS file. With CSS, we could control the style of certain elements, and also create classes to attach to multiple elements. My website includes an intro to Belugas, a photo gallery, some facts, and video of the baby beluga song.

Environmental Call to Action

For our project calling action to an environmental issue, I chose to focus on Light Pollution. I continued to experiment with different styles in CSS. I set a background image to the header and added a rounded border around it. I also added a starry background image to the whole page. We learned how to add a hover function to elements in CSSm which I used to display the information on Light Pollution.

Collector's Guide

I worked with Amy on this project to show some different fashion styles. We made different html pages for each style, and used anchors to link between pages. We also conducted a survey to see which style was the omst popular among AWS students and faculty.

Guide to the PNW

This next project used much more complex CSS functions like animation, hovers, image settings, etc. This page share some fun activities to do in the area during the springtime. There is a Nature section with three activities to enjoy Washington's nature, and an Entertainment section with some indoor activities.

Buzzfeed Quiz

I worked with Jaein on the project to create a buzzfeed style quiz that tells your which AWS Teacher you are most like. There are six questions that will determine whether you are most like Sidman, Considine, Barber, etc. We used javascript to make the quiz work. This project also uses two types of animations.

The Four Color Theorem

This is the newest website I made just for this webpage. It discusses what the Four Color Theorem is, which we learned about earlier in the semester. It contains information and background on the Theorem, and also includes a video and some coloring pages to try the theorem yourself.
